# Testing Strategy

## Tools
- **Jest**: Core test runner (provided by `react-scripts`).
- **React Testing Library**: Used for component testing.
- **@testing-library/jest-dom**: Custom matchers for DOM state.

## Current State
- **Scripts**: `npm test` runs the testing suite in watch mode.
- **Coverage**: Basic setup exists, but the codebase lacks extensive unit or integration test coverage in visible directories.
- **Focus**: Future focus should be on critical paths like Checkout and Payment processing.
