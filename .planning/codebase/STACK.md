# Technology Stack

## Core
- **Framework**: React 18.3.1
- **Language**: JavaScript (JSX)
- **Build Tool**: Create React App (react-scripts)
- **Environment**: Node.js

## State Management
- **Redux**: version 4.1.2
- **Middleware**: Redux Thunk
- **DevTools**: Redux DevTools Extension

## Styling & UI Libraries
- **Tailwind CSS**: Utility-first styling (v3.0.15) - **Primary Standard**
- **Material UI (MUI)**: Layouts, icons, and data grids - **Core Component Library**
- **Emotion**: Dependency for MUI and CSS-in-JS
- **Styled Components**: Legacy component-level styling (to be migrated to Tailwind)
- **Deprioritized/Removed**: Ant Design (antd), React Bootstrap (removed to reduce bloat)

## Navigation & Routing
- **React Router DOM**: Client-side routing (v6.2.1)

## API & Data Fetching
- **Axios**: HTTP client with global defaults configured in `App.js`
- **JS Cookie**: Handling cookies for authentication

## Animations & Effects
- **Framer Motion**: Page transitions and element animations
- **GSAP**: High-performance animations
- **Lottie**: Vector-based animations via `dotlottie-react`
- **Typed.js**: Typing effects

## Utilities
- **Moment.js**: Date formatting
- **React Helmet Async**: SEO and meta tag management
- **React Hot Toast / Notistack**: Toast notifications
- **React Loading Skeleton**: Loading states
- **React Quill**: Rich text editor
- **React Slick**: Carousel/sliderr functionality
- **Three.js**: 3D rendering capabilities
- **WebFontLoader**: Font loading management
